## 🔄 User Journey Flows

### Flow 1: Browse → Order Service
```
Home → Services → Click Service Card → View Details Modal → 
Order Button → Order Form → Submit to WhatsApp
```

### Flow 2: Calculate Price → Contact
```
Home → Scroll to Calculator → Select Casket → 
Add Services → See Total → Contact Button → 
Scroll to Contact → WhatsApp
```

### Flow 3: View Gallery → Order
```
Home → Gallery → Filter Category → View Image → 
Lightbox → Close → Scroll to Products → 
Select Product → Order Form → WhatsApp
```

### Flow 4: Read FAQ → Contact
```
Home → FAQ → Read Questions → Expand/Collapse → 
FAQ CTA Button → WhatsApp
```

### Flow 5: Quick Contact
```
Home → Hero → Hubungi Kami Button → 
Scroll to Contact → WhatsApp Button
```
