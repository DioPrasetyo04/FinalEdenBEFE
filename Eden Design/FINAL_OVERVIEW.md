## 🎨 Design System

```

Secondary Blue:  #DCEAF5
Background:      #FFFFFF
Card:            #F9FAFB

```

**Dark Mode:**
```
Primary Gold:    #D4AF37
Background:      #0F172A
Card:            #1E293B
Surface:         #334155
Text Primary:    #F9FAFB
Text Secondary:  #CBD5E1
Border:          #334155
```

### Typography
- **Headings:** Playfair Display (serif, elegant)
- **Body:** Inter (sans-serif, clean)
- **Hierarchy:** H1 (3rem) → H2 (2.25rem) → H3 (1.5rem) → Body (1rem)

### Animations
- **Scroll Animations:** Fade in, stagger delays
- **Hover Effects:** Lift, scale, glow, rotate
- **Transitions:** 200ms ease-in-out
- **Counters:** 2s smooth counting
- **Modal:** Spring animation
- **Loading:** 2s duration
